## About

Between 2001 and 2015, Bolivia dramatically expanded social spending. This book evaluates six major programs using rigorous causal methods:

| Program | Method | Key Finding |
|---------|--------|-------------|
| PLANE (emergency employment) | Propensity Score Matching | Stabilized consumption; zero effect on future employability |
| MPED (youth job training) | Fuzzy RDD | +15.6pp formal employment at 6 months — vanished by 18 months |
| Renta Dignidad (universal pension) | DiD + Changes-in-Changes | Increased income; significantly reduced female labor supply |
| Bono Juancito Pinto (school subsidy) | Structural model | Enrollment effect only for ages 6–8; zero for older children |
| Yo Sí Puedo (literacy campaign) | ITS + PSM | No significant reduction in functional illiteracy despite 824K certifications |
| BDP microcredit | PSM | Strong effects in manufacturing; zero in agriculture |
